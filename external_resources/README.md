# External Resources for AI-Based Proctoring System

This folder contains all external resources and dependencies used by the proctoring system to ensure nothing gets lost.

## Contents:

### 1. OpenCV Haar Cascades
- `haarcascade_frontalface_default.xml` - Face detection
- `haarcascade_eye.xml` - Eye detection

### 2. Original Gaze Tracking Files
- `face_eye_tracker.py` - Original gaze tracking script
- `templates/index.html` - Original web interface

### 3. Python Dependencies
- `requirements.txt` - All required Python packages

### 4. Configuration Files
- `start_proctoring_system.bat` - Startup script
- Various configuration files

## Usage:
All resources are self-contained in this project. No external downloads required.
